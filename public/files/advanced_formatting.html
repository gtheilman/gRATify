Advanced Formatting

This page shows plain-text examples you can paste into the question or answer fields in the assessment editor.
The app supports Markdown, KaTeX, and AsciiMath.

Markdown: https://en.wikipedia.org/wiki/Markdown
KaTeX: https://katex.org/
AsciiMath: https://asciimath.org/

Markdown Examples

Use these in question stems or answer choices. Each example shows the plain text first, then the rendered output.

KaTeX Examples

Wrap inline math in single dollar signs and display math in double dollar signs.

Plain text
Inline LaTeX: $E = mc^2$
Rendered
Inline LaTeX: $E = mc^2$
Plain text
Display LaTeX:
$$a^2 + b^2 = c^2$$
Rendered
Display LaTeX:
$$a^2 + b^2 = c^2$$
Plain text
Fractions and roots:
$$\frac{y_2 - y_1}{x_2 - x_1}$$
$$\sqrt{16} = 4$$
Rendered
Fractions and roots:
$$\frac{y_2 - y_1}{x_2 - x_1}$$ $$\sqrt{16} = 4$$

AsciiMath Examples

Wrap AsciiMath expressions with @ symbols.

Plain text
Inline AsciiMath: @sqrt(16)=4@
Rendered
Inline AsciiMath: @sqrt(16)=4@
Plain text
Display AsciiMath:
@int_0^1 x^2 dx@
Rendered
Display AsciiMath:
@int_0^1 x^2 dx@
Plain text
Another example:
@sum_(i=1)^n i = n(n+1)/2@
Rendered
